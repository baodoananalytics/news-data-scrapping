# news-data-scrapping
Automated news extraction by BeautifulSoup for web scraping and Power Automate for daily delivery to communication channels

# Overview
Developed an automated news aggregation system to provide daily updates on the Vietnamese Real Estate and Construction Materials sectors. The pipeline utilizes Requests and BeautifulSoup for web scraping (source, title, URL, summary), Pandas for structured data management in Excel, and Power Automate to trigger automated delivery to chatbox platforms.
